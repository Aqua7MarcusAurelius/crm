# Стек технологий

## Язык
- **TypeScript** — весь проект (бэкенд + фронтенд)

## Backend
| Технология | Назначение |
|---|---|
| NestJS | Фреймворк, API, бизнес-логика |
| Prisma 7 | ORM для PostgreSQL |
| Zod 4 | Валидация данных |
| Bcrypt | Хеширование паролей |
| JWT (Passport) | Аутентификация |
| BullMQ | Очереди фоновых задач (через Redis) |
| Nodemailer | Отправка email |
| Pino | Логирование |
| Socket.io | WebSocket (чат, real-time) |

## Frontend
| Технология | Назначение |
|---|---|
| Next.js 16 | Фреймворк (App Router) |
| React | UI |
| Tailwind CSS v4 | Стили |
| shadcn/ui | UI-компоненты |
| Zustand | Стейт-менеджмент |
| TanStack Query | Загрузка данных, кеш |
| React Hook Form | Формы |
| PapaParse | Парсинг CSV |

## Инфраструктура
| Технология | Назначение |
|---|---|
| Docker Compose | Оркестрация сервисов |
| PostgreSQL 16 | Основная БД |
| Redis 7 | Кеш, очереди |
| MinIO | Файловое хранилище (S3-совместимое) |

## Инструменты
| Инструмент | Назначение |
|---|---|
| Git + GitHub | Версионирование |
| VS Code Remote SSH | IDE |
| Prisma Studio | GUI для БД |
| nvm | Управление версиями Node.js |
